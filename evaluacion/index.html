1.¿Que es una variable?

Es un espacio en la memoria de una computadora que se utiliza para almacenar datos que pueden cambiar durante la ejecución de un programa.

2.¿Que es una costante?

Es un valor de tipo permanente que no puede ser modificado durante la ejecución de un programa.

3.Mencione los tipos de datos y definicion.

Son categorías que definen el tipo de valor que puede almacenar una variable y las operaciones que se pueden realizar con ella.

Tipos de datos comunes.

Numéricos:

•Enteros (Integer):

Almacenan números sin decimales (ej. 30, -5). Se usan para contar.

•Coma flotante (Float/Double):

Almacenan números con decimales (ej. 19.99, 3.14).

•Flotante de precisión simple:

Un tipo para cálculos con decimales.

Cadenas de texto (String):

•(String):

Secuencias de caracteres, como texto o nombres, que se encierran entre comillas (ej. "Ana", "Hola mundo").

4.¿Que son datos estructurados y no estructurados?

Los datos estructurados tienen un formato fijo y predefinido, organizados en tablas con filas y columnas, mientras que los datos no estructurados carecen de un esquema fijo y pueden ser de cualquier tipo, como texto, imágenes o audio.

5.Diferencia entre datos e información

La diferencia principal es que los datos son hechos brutos y sin procesar, mientras que la información es el resultado de organizar, procesar e interpretar esos datos para darles significado y utilidad.

6.¿Que es una API? y de su ejemplo

conjunto de protocolos y herramientas que permite que dos aplicaciones de software se comuniquen entre sí. Un ejemplo seria los; servicios de mapas: al usar una aplicación como Google Maps para buscar direcciones, la aplicación utiliza la API de Google para mostrar mapas, obtener datos de ubicación y calcular rutas.

7.¿Que son los manejadores de base de datos?

Son un conjunto de programas de software que permiten a los usuarios y aplicaciones crear, gestionar y administrar la información almacenada en una base de datos.

8.¿Que son contenedores docker?

Los contenedores Docker son instancias en ejecución de una imagen Docker, lo que significa que son unidades estandarizadas y portátiles que empaquetan una aplicación con todas sus dependencias (código, bibliotecas, herramientas) para que pueda ejecutarse de manera consistente en cualquier entorno.

9.¿Como ejecutar un contenedor en docker? y de un ejemplo

Para ejecutar un contenedor en Docker, se usa el comando "docker run" seguido del nombre de la imagen, como en "docker run ubuntu:24.04". Puedes agregar opciones para controlarlo, por ejemplo, "-d" para ejecutarlo en segundo plano, "-p" para mapear puertos, o "--name" para darle un nombre único. Para interactuar con un contenedor ya en ejecución, se usa "docker exec -it < nombre_contenedor >/bin/bash".

10.Dedifina WWW, IP, DNS ISP,VPN y HTTP

•Definicion de WWW:

WWW son las siglas de World Wide Web, que en español significa "Red Mundial de Información". Se refiere a un sistema de páginas web interconectadas accesibles a través de Internet, y aunque a menudo se usa como sinónimo, es un servicio distinto al propio Internet. El uso de "www" en una dirección web (URL) es una convención por la cual se identifica que el destino es una página de la web.

•Definicion de IP:

IP significa Protocolo de Internet y se refiere tanto al protocolo de comunicación de Internet como a una dirección numérica única que se asigna a cada dispositivo conectado a una red. Su función principal es identificar y ubicar los dispositivos para permitir el envío y la recepción de datos entre ellos.

•Definicion de DNS:

DNS son las siglas de Sistema de Nombres de Dominio (Domain Name System), un sistema que funciona como la agenda telefónica de Internet, traduciendo nombres de dominio fáciles de recordar (como www.google.com) a direcciones IP numéricas que las computadoras necesitan para conectarse. Cuando escribes una URL en tu navegador, el DNS encuentra la dirección IP correcta para poder cargar la página web.

•Definicion de ISP:

ISP significa Proveedor de Servicios de Internet (del inglés Internet Service Provider), y es la empresa que proporciona acceso a Internet a usuarios y organizaciones. Estas empresas conectan a sus clientes a la red global a través de diversas tecnologías como fibra óptica, cable o ADSL, y a menudo ofrecen servicios adicionales como correo electrónico o alojamiento web.

•Definicion de VPN:

Una VPN (Red Privada Virtual) es un servicio que crea una conexión segura y cifrada entre tu dispositivo e Internet. Su función principal es enmascarar tu dirección IP y proteger tus datos personales al crear un "túnel" digital, haciendo que tu actividad en línea sea más privada y segura. Esto es útil para navegar de forma anónima, evitar restricciones geográficas, y proteger tu información en redes Wi-Fi públicas.

•Definicion de HTTP:

HTTP significa Protocolo de Transferencia de Hipertexto (en inglés, Hypertext Transfer Protocol) y es el protocolo fundamental para la comunicación en la World Wide Web. Funciona con un modelo de solicitud-respuesta: cuando navegas a un sitio web, tu navegador (el cliente) envía una solicitud HTTP al servidor, y este último responde enviando la información solicitada, como archivos HTML.
